    **Step 3: Capture portrait during conversation**

    Add a `useEffect` that fires a portrait capture once the visitor has been in the conversation screen for a few seconds (face is facing the tablet). Use a 5-second delay after entering conversation screen:
    ```tsx
    const portraitCapturedRef = useRef(false);
    useEffect(() => {
      if (screen === 'conversation' && !portraitCapturedRef.current && POS_SERVER_URL) {
        const timer = setTimeout(() => {
          portraitCapturedRef.current = true;
          void capturePortrait();
        }, 5000);
        return () => clearTimeout(timer);
      }
      // Reset flag when leaving conversation
      if (screen === 'sleep') {
        portraitCapturedRef.current = false;
      }
    }, [screen, capturePortrait]);
    ```

    - If POS_SERVER_URL is empty/not configured, portrait capture silently skips (no errors)
