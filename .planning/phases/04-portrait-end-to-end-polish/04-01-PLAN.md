<objective>
Create the portrait capture pipeline: upgrade camera resolution, share the videoRef between face detection and portrait capture, create the usePortraitCapture hook (Canvas drawImage + toBlob + FormData POST), and wire the capture/upload trigger in App.tsx.

Purpose: R7 requires the visitor's face to be captured from the tablet camera during conversation and sent to the POS server for dithered thermal printing. The POS server already has the full portrait pipeline (style transfer, face landmarks, zoom crops, dithered printing). The tablet's only job is: capture a high-quality JPEG frame, POST it.

Output: Working portrait capture pipeline from camera frame to POS server upload, with unit tests.
