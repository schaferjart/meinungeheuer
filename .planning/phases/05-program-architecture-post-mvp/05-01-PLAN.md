3. `ResultDisplay` type: `'aphorism' | 'definition' | 'raw_transcript'` -- how the result shows on screen.

4. `PromptParams` interface: `{ term: string; contextText: string | null; language: string }` -- the generic params all programs receive for prompt building.

5. `ConversationProgram` interface with fields:
   - `id: string` -- unique identifier stored in installation_config.program
   - `name: string` -- human-readable
   - `description: string`
   - `stages: StageConfig` -- which pipeline stages are active
   - `buildSystemPrompt: (params: PromptParams) => string` -- builds the full system prompt
   - `buildFirstMessage: (params: PromptParams) => string` -- builds the agent's opening line
   - `printLayout: PrintLayout` -- which print template to use
   - `resultDisplay: ResultDisplay` -- how to show the result
   - `sessionMode: Mode` -- the Mode value for Supabase session records

Import `Mode` from `../types.js`.
