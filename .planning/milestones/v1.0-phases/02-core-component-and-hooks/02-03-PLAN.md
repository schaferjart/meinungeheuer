# Plan 03: useAutoScroll Hook — Comfort Zone Scrolling

## Objective
Implement the `useAutoScroll` hook that auto-scrolls a container to keep the active word within a comfort zone (20%-65% viewport band), with a 3-second cooldown after manual scroll.

## Context
The existing `TextReader.tsx` has this logic: comfort zone check via `getBoundingClientRect()`, scroll to 35% mark when word is outside zone, manual scroll detection via `onscroll` listener, and cooldown timer. Extract as a standalone hook. The hook takes `containerRef`, `activeWordIndex`, `wordElementsRef`, and config options.

## Tasks

<task id="03-01">
### Implement useAutoScroll hook
Create `src/hooks/useAutoScroll.ts`. Accept `{ containerRef, activeWordIndex, wordElementsRef, comfortTop?: number (default 0.20), comfortBottom?: number (default 0.65), cooldownMs?: number (default 3000), enabled?: boolean (default true) }`. Implementation:
- `useEffect` on `activeWordIndex`: get active word span from `wordElementsRef`, call `getBoundingClientRect()` on both span and container, if word is above `comfortTop` fraction or below `comfortBottom` fraction of container height, scroll to bring word to 35% mark.
- Manual scroll detection: attach `scroll` event listener to container, record timestamp on scroll. Suppress auto-scroll for `cooldownMs` after last manual scroll.
- Cleanup: remove scroll listener, clear timeout.
